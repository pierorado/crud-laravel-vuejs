

# Gestion de Estudiantes

En este proyecto se puede registra, modifica, eliminar y visualiza los datos de estudiantes.


## Tech Stack

**Client:** Vue y  Bootstrap [click aqui](https://github.com/pierorado/CRUD-Front-LaravelVuejs) 

**Server:** Laravel
